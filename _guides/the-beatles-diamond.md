## About the Table
The Beatles pinball machine by Stern (2018) is based on the classic Seawitch layout but updated with Beatles-themed modes and features. The Diamond Edition is a limited, numbered collectible with mirrored backglass. The game focuses on the band's early years and Beatlemania era, featuring classic electromagnetic sound effects and retro aesthetics while maintaining modern Stern gameplay depth.

## Skill Shots
There are four available skill shots with varying risk/reward ratios:

- **10K Short Plunge**: Easiest skill shot to the star rollover at the top
- **25K Medium Plunge + Loop**: Medium plunge, then shoot the loop with the upper flipper - **this is the best risk/reward option** and often helps with mode progress
- **100K Gap Shot**: Plunge into the small hole below the magnet - high risk but decent reward
- **100K Drop Target**: Full plunge then use upper left flipper to hit the 1964 drop targets - difficult timing

The 25K loop skill shot is recommended as it's reliable and frequently contributes to mode objectives.

## Early Game Strategy
- **Always select your mode before plunging** using the flipper buttons - modes start automatically with each ball
- **Loop Mania is an excellent first mode choice** as the loop skill shot provides immediate progress
- Focus on completing your selected mode's objective for solid base scoring
- Make safe, controlled shots to build mode progress rather than risky attempts at drop target banks
- Establish ball control early as the game rewards precision over wild play

## Mid Game Strategy
- **Complete modes to light Super Jackpots** behind the lower left FAB drop targets - these re-collect your entire mode score
- **Stack Super Jackpots** for massive scoring: complete multiple modes without collecting the Super Jackpot, then collect for 2X, 3X, or higher multipliers on all stacked modes
- **Complete FAB and FOUR drop target banks, then shoot the loop** to start a new mode
- Target the right spinner when John, Paul, George, and Ringo targets are completed to increase spinner value
- Work toward completing all drop target banks (FAB, FOUR, 1964) on a single ball to light extra ball at the inlanes

## Late Game Strategy
- **If approaching your fifth mode before Beatlemania**, consider timing out the mode ("Z-Mac" strategy) rather than risking a drain - Beatlemania wizard mode is worth significantly more
- Focus on bonus multiplier collection through drop targets if you have a long, controlled ball
- **On Ball 3 specifically**: Select "All My Loving" mode before plunging for an automatic 2-ball multiball
- Be conservative with risky shots late in the ball - the game rewards consistency over hero shots

## Multiballs
- **All My Loving Multiball**: Only available on Ball 3 - select this mode before plunging for immediate 2-ball multiball
- **Hard Day's Night/Beatlemania**: Complete all 5 song modes to start this 4-ball wizard mode multiball with increased jackpot values
- **Taxman Multiball**: Additional multiball available (specific start conditions need verification)
- During multiballs, focus on ball control and safe jackpot shots rather than trying to keep all balls in play

## Key Tips
- Super Jackpots will not light if you make little to no shots during a mode - complete mode objectives properly
- FAB drop targets reset immediately after a song mode ends - don't set them up too early
- The loop magnet grabs most shots during certain modes to feed the pop bumpers
- BEATLEMANIA letters can be collected during pop bumper modes for awards like 2X spinners
- Bonus collection and bonus multiplier from drop targets can be valuable on long balls
- The game uses Seawitch-based rules but focus on the Beatles-specific scoring rather than legacy Seawitch strategies
- Completing all three drop target sets (FAB, FOUR, 1964) on one ball lights extra ball at inlanes
